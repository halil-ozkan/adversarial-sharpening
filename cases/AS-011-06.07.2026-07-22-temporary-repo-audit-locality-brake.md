## Minimal Repro

```text
User:
Audit my repo for the time being.

Assistant:
Treats the repo as if it is being evaluated as a finished public product, expands into broad release-readiness analysis, and makes the conversation feel suffocating.
```

## Observed Failure

The assistant missed the small cue:

```text
for the time being
```

That phrase should have acted as a locality brake.

Instead of auditing the current temporary state, the assistant analyzed the work as if the user were asking for a finished-product review.

## Expected Behavior

A better assistant should preserve the temporary scope first:

```text
Understood: temporary/current-state audit, not final release judgment.
```

Then it should audit only the live state:

```text
What is true right now?
What is blocking publication right now?
What should not be overinterpreted yet?
What is the smallest useful patch?
```

It should not unfold the whole public-product roadmap unless the user asks.

## Why It Matters

Small temporal cues change the job.

Missing this cue creates practical friction:

- misreads user intent
- over-expands the task
- turns a staging check into a final verdict
- makes the user fight the assistant's momentum
- suffocates a conversation that needed a narrow audit

## Severity

Medium.

The likely harm is not safety-critical, but it damages workflow trust and drains energy. It is especially costly in high-context artifact work where the difference between staging and release is the whole point.

## Boundary Cases

Pass case:

```text
The assistant explicitly treats "for the time being" as a temporary/current-state constraint and keeps the audit narrow.
```

Fail case:

```text
The assistant treats the repo as a finished public product and expands into broad release analysis without preserving the temporary cue.
```

Ambiguous case:

```text
The user asks for a full publication audit, launch review, or final readiness review. In that case, broad release analysis is appropriate.
```

## Mitigation

Locality-brake rule:

```text
When a user says "for now," "for the time being," "staging," "temporary," "private," or "not public," preserve the current-state boundary before analyzing larger release implications.
```

Short form:

```text
Temporary cue first. Final-product frame second.
```
